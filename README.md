🌾 CropIQ — Intelligent Crop Recommendation System
====================================================

A farmer-first web application that recommends the top 3 best crops
based on real field conditions — built with a pure rule-based scoring
engine. No internet required. No app to install. Just open and use.

🔍 How It Works
----------------
Farmers input 7 field conditions:
  • Soil type (Loamy / Sandy / Clay)
  • Water availability (Low / Medium / High)
  • Season (Kharif / Rabi / Summer)
  • Temperature, Rainfall, Humidity, Soil pH

The engine scores all 14 crops across 7 weighted factors and returns
the Top 3 matches with confidence percentages and farming action steps.

✨ Features
-----------
  ✅ Visual tap-based UI — no reading or typing required
  ✅ Sliders with emoji guides for uneducated farmers
  ✅ 6 Indian language toggle (full translation)
  ✅ Step-by-step farming instructions per crop
  ✅ Export results as PDF (print dialog)
  ✅ Export results as PNG image (share on WhatsApp)
  ✅ Animated farm environment (sky, sun, birds, hills)
  ✅ 100% offline — single HTML file, zero dependencies

🧠 Scoring Engine
------------------
  Factor         Weight
  ─────────────────────
  Temperature     20%
  Rainfall        20%
  Soil Type       15%
  Soil pH         15%
  Humidity        10%
  Water Supply    10%
  Season          10%

📦 Tech Stack
--------------
  • Vanilla HTML / CSS / JavaScript
  • HTML5 Canvas (PNG export)
  • Google Fonts (Nunito, Noto Sans regional scripts)
  • Zero frameworks. Zero dependencies. Zero server.

🚀 Getting Started
-------------------
  1. Download cropiq-v3.html
  2. Open in any browser
  3. Select your language
  4. Fill in your field conditions
  5. Get your crop recommendation instantly

🎯 Built For
-------------
  Real farmers across India — especially those with limited literacy,
  limited internet access, and limited time. Designed to be used on
  a basic Android phone with one hand in a field.
